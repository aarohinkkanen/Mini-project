# Mini-project

Smart Ansible Server Monitoring

Automaattinen Linux‑palvelimen valvontajärjestelmä, joka asentuu yhdellä komennolla Ansiblella.

- **Tekijät:** Minttu Hillebrandt & Aaro Hinkkanen
- **Lisenssi:** GNU General Public License v3.0 (GPL‑3.0)

> kuva

# Projektin tarkoitus

Projektin tavoitteena on tarjota kevyt, automatisoitu ja helposti toistettava palvelinvalvontaratkaisu, joka seuraa CPU‑, RAM‑ ja levytilan käyttöä ja kirjaa varoitukset lokiin.
> Ansible hoitaa koko asennuksen automaattisesti, joten valvonta voidaan ottaa käyttöön useilla palvelimilla yhdellä komennolla.
